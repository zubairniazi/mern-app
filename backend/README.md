# Workout Buddy APIs

### Install Dependencies
```
npm i
```

### Configure Environments
Create `.env` file and add below content:

```
PORT=5000
MONGO_URI=YOUR_MONGO_DB_URI
CORS_ORIGIN=
```

Note: The value of `CORS_ORIGIN` will be client's app URL like `example.com`.

### Run Server
```
npm run dev
```
